What is a responsive web design?
	HTML5 and CSS3 provides a kind of design of the page, responsive web design

	it is a design where the page content presentation will be changed as per the device size.

	@media-queries  is the concept of creating css classes using CSS3


What is a Bootstrap?
	It is a plug-in / tool where it allows to design web pages for the responsive format. 
		It will provide   predefined css classes and js methods. 
To learn Bootstrap, we can use "getbootstrap.com" site. it is a official website for the bootstrap.


<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">

<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>

for page design using bootstrap, we have to follow below given 4 things

1. ViewPort 
2. BreakPoints
	Breakpoints are customizable widths that determine how your responsive layout behaves across 		device or viewport sizes in Bootstrap.
		i.e. identifing device width.
3. Containers
	Containers are a fundamental building block of Bootstrap that contain, pad, and align your 		content within a given device or viewport.

4. Grid System
Use our powerful mobile-first flexbox grid to build layouts of all shapes and sizes thanks to a twelve column system, six default responsive tiers, Sass variables and mixins, and dozens of predefined classes.
